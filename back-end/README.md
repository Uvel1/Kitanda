# Back-End — Kitanda 2.0

Esta pasta será destinada ao desenvolvimento do servidor e da base de dados da plataforma Kitanda.

## Tecnologias previstas
- **Linguagem:** Python
- **Base de Dados:** SQLite (início) → PostgreSQL (produção)
- **Autenticação:** A definir pelo grupo

## Estrutura futura
```
back-end/
├── app.py              # Ficheiro principal do servidor
├── models/             # Modelos da base de dados
├── routes/             # Rotas e endpoints da API
├── controllers/        # Lógica de negócio
└── database/           # Ficheiros da base de dados
```
