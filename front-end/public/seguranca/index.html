Segurança

Dicas e políticas para garantir transações seguras.

O nosso compromisso

Na Kitanda, a segurança dos nossos utilizadores é a nossa prioridade absoluta. Implementamos medidas tecnológicas e políticas rigorosas para garantir que todas as transações, dados pessoais e interações ocorram num ambiente protegido.

Proteção de Dados

  • Encriptação: Todas as passwords são encriptadas de forma segura. Não armazenamos palavras-passe em texto limpo.
  • Comunicações: Utilizamos JWT (JSON Web Tokens) para manter as sessões autenticadas com os mais altos padrões de segurança.
  • Privacidade: Os seus dados pessoais não são partilhados com terceiros sem o seu consentimento explícito, conforme os nossos termos.

Dicas de Segurança para Compras e Vendas

  • Nunca partilhe a sua palavra-passe ou códigos de verificação com terceiros.
  • Evite fazer pagamentos fora das plataformas bancárias oficiais de Angola (Multicaixa Express, Transferências IBAN).
  • Em caso de encontrar anúncios suspeitos ou atividades fraudulentas, utilize o botão "Denunciar" imediatamente.
